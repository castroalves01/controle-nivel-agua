# Controle de Nível de Água

![Status](https://img.shields.io/badge/status-concluído-brightgreen)

Sistema simples em Python que simula o monitoramento de um reservatório de água, exibindo diferentes níveis com cores no terminal.

---

## Funcionalidades

- Exibição dos níveis do reservatório  
- Cores diferentes para cada situação  
- Simulação no terminal
